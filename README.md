# shadow

=======
1. Descripción del proyecto

Este proyecto consiste en una página web desarrollada con HTML, CSS y JavaScript, cuyo objetivo es demostrar el uso de estructuras semánticas, estilos modernos y funcionalidades básicas.

Incluye ejemplos de:

Estructura HTML
Diseño con CSS (Flexbox y Grid)
Interactividad con JavaScript
    2. Objetivo
Aplicar etiquetas HTML semánticas
Usar propiedades CSS modernas
Implementar diseño responsive
Agregar interactividad básica
    3. Tecnologías utilizadas
HTML5
CSS3
JavaScript
    4. Estructura del código
    HTML
<header>

Contiene el título principal y la barra de navegación.

<nav>

Permite navegar entre secciones mediante enlaces.

<main>

Contiene todo el contenido principal de la página.

<section>

Divide el contenido en bloques organizados.

<article>

Representa contenido independiente.

<aside>

Muestra información secundaria o adicional.

<footer>

Contiene información final como derechos de autor.

    Elementos adicionales
Listas (<ul>, <li>)

Muestran información en formato de lista.

Tabla (<table>)

Organiza datos en filas y columnas.

Formulario (<form>)
